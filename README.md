A hands-on, virtualized cybersecurity environment designed for learning, experimentation, and testing security tools and techniques in a safe, isolated setup.

This project demonstrates how to build and configure a multi-virtual machine (multi-VM) cybersecurity lab using common virtualization platforms such as VirtualBox, VMware Workstation. The lab simulates real-world networked systems, virtual lab setup, network configuration, installing of core security tools, allowing you to practice ethical hacking, incident response, malware analysis, and system hardening.
